##  Overview

Tileset Manager is a lightweight tool to import and preview tilesets before exporting or slicing them for 2D game engines such as Unity or Godot.

This first version includes UI, drag-and-drop import, and a clean layout.

---

##  Features (v1)

-  Drag & drop tileset import  
-  File chooser support  
-  Clean and responsive UI  
-  Built with React 18 + Vite  
-  Dark theme  
-  Project structured for future slicing & export features  

---

##  Installation

### 1. Clone the project

```bash
git clone https://github.com/ComiccSanss2/tileset-manager.git
```

### 2. Install dependencies

```bash
cd tileset-manager
npm install
```

### 3. Start the dev server

```bash
npm run dev
```
