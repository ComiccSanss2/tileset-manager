##  Overview

PIXENO is a lightweight, fast and fully client-side tileset manager designed for 2D game developers.

This tool helps you:

Analyze tiles

Slice tilesets

Add collision metadata

Export clean assets for engines like Unity and Godot

Everything runs locally in your browser: no uploads, no backend, no tracking.

---

##  Features (v1)

-  Drag & drop tileset import  
-  File chooser support  
-  Clean and responsive UI  
-  Built with React 18 + Vite  
-  Dark theme  
-  Project structured for future slicing & export features

##  Installation

### 1. Clone the project

```bash
git clone https://github.com/ComiccSanss2/tileset-manager.git
```

### 2. Install dependencies

```bash
cd tileset-manager
npm install
```

### 3. Start the dev server

```bash
npm run dev
```
